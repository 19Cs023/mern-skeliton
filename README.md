# MERN skeliton 

This is a full-stack MERN (MongoDB, Express, React, Node.js) app

## Project Structure

- `frontend/` - React frontend powered by Vite and Tailwind CSS.
- `backend/` - Node.js/Express backend API connected to MongoDB.

## Getting Started

### Backend
1. Navigate to `backend/` folder: `cd backend`
2. Run backend in development mode: `npm run dev`
*(Requires MongoDB running locally on port 27017, or set the MONGO_URI environment variable).*

### Frontend
1. Navigate to `frontend/` folder: `cd frontend`
2. Run frontend in development mode: `npm run dev`

## Dependencies
- **Frontend**: React, React Router DOM, Axios, Tailwind CSS, Vite
- **Backend**: Express, Mongoose, Cors, Dotenv, Nodemon
